# gitcodetest

1. 查看项目的分支们(包括本地和远程) 
命令行 : $ git branch -a
2. 删除本地分支 
命令行 : $ git branch -d <BranchName>
3. 删除远程分支 
命令行 : $ git push origin --delete <BranchName>
4. 合并分支
git merge dev
5. 切换远程分支 
git checkout -b second-period origin/second-period
6. git fetch
7. git branch -va
